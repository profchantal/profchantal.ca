VERSION SITE PROFCHANTAL.CA - V3

Cette version corrige la cohérence des pages :
- deux livres seulement sont affichés;
- le troisième livre non publié a été retiré de l'accueil, de la page Livres, de la page Ressources et de la page À propos;
- les menus sont harmonisés sur toutes les pages;
- les chemins CSS et JavaScript sont cohérents;
- les noms d'images sont descriptifs et stables;
- le cache est forcé avec ?v=3.

À téléverser à la racine du site :
- index.html
- style.css
- pages/
- assets/
- NOMS_FICHIERS_IMAGES_SITE.txt

Après téléversement, faire Ctrl + F5 dans le navigateur.
